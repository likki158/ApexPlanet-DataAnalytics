# ApexPlanet Data Analytics Internship - Final Project Summary

## Internship Overview
This project portfolio showcases the successful completion of multiple data analytics tasks involving data cleaning, exploratory data analysis, dashboard development, and statistical validation using real-world e-commerce data.

---

# Tasks Completed

## Task 1 - Data Immersion & Wrangling
- Cleaned raw e-commerce dataset
- Handled missing values and duplicates
- Converted datetime columns
- Engineered new features
- Generated cleaned dataset

## Task 2 - Exploratory Data Analysis & Business Intelligence
- Performed univariate and multivariate analysis
- Identified business insights
- Created trend and KPI visualizations
- Built SQL-style business queries

## Task 3 - Interactive Dashboarding
- Developed Power BI dashboard
- Created KPI cards and slicers
- Visualized delivery and order trends
- Built interactive business intelligence dashboard

## Task 4 - Data Storytelling & Statistical Validation
- Performed hypothesis testing
- Conducted T-Test analysis
- Validated delivery performance insights
- Generated business storytelling conclusions
