# Task 2 - Exploratory Data Analysis & Business Intelligence

## Objective
The objective of this task was to perform exploratory data analysis and generate business insights from the cleaned e-commerce dataset.

## Analysis Performed
- Order status analysis
- Monthly order trend analysis
- Delivery time distribution
- Late delivery analysis
- Correlation analysis
- KPI identification
- Business question analysis

## Key Insights
- Most orders were delivered successfully.
- Late deliveries were relatively low.
- Delivery performance varied across months.
- Average delivery time was approximately 12 days.
- Certain months experienced operational delays.

## Tools Used
- Python
- Pandas
- Matplotlib

## Outcome
The analysis provided valuable insights into logistics performance, customer order behavior, and operational efficiency.
