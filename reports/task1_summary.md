# Task 1 - Data Immersion & Wrangling

## Objective
The objective of this task was to clean and prepare the Olist E-Commerce dataset for analysis.

## Steps Performed

### 1. Data Loading
Loaded multiple datasets including:
- Customers
- Orders
- Order Items
- Payments
- Reviews
- Products
- Sellers

### 2. Data Inspection
Performed:
- shape analysis
- datatype inspection
- missing value analysis
- duplicate checking

### 3. Data Cleaning
- Converted timestamp columns to datetime format
- Verified duplicate records
- Investigated missing values

### 4. Feature Engineering
Created new columns:
- delivery_time_days
- purchase_year
- purchase_month
- purchase_day
- is_late_delivery

## Key Insights
- Average delivery time was approximately 12 days.
- Maximum delivery time observed was 209 days.
- Late deliveries identified: 7,827 orders.

## Output
Cleaned dataset exported successfully for future analysis.
