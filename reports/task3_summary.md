# Task 3 - Deep-Dive Analysis & Interactive Dashboarding

## Objective
The objective of this task was to build an interactive business intelligence dashboard using Power BI to analyze e-commerce delivery and order performance.

## Dashboard Features
- Total Orders KPI
- Average Delivery Time KPI
- Monthly Order Trend Analysis
- Late vs On-Time Delivery Analysis
- Interactive Month Filter (Slicer)

## Business Insights
- Most orders were delivered successfully.
- Late deliveries represented a smaller percentage of total deliveries.
- Certain months showed increased delivery delays.
- Order activity varied across different months.

## Tools Used
- Power BI
- Python
- Pandas

## Outcome
An interactive dashboard was successfully created to monitor operational performance and support business decision-making.
